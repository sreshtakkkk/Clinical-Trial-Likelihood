# Datasets Directory

Place your datasets here:
- `raw_data.csv`   — Main clinical trial dataset (nctid, status, why_stop, label, phase, diseases, icdcodes, drugs, smiless, criteria)
- `clintox.csv`    — ClinTox toxicity dataset (smiles, FDA_APPROVED, CT_TOX)

The `processed/` folder is auto-created by the preprocessing pipeline.
